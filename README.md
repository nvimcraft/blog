# blog

Personal blog.

## Stack

### Client

- Vite (rolldown-vite)
- React
- React Compiler
- TypeScript
- TanStack Router

### Server

- Cloudflare Worker (one-worker: serves SPA + `/api/*`)
- Supabase
- PostgreSQL
- Supabase JS client
- Resend (newsletter emails)

### Tooling

- pnpm
- Prettier
- ESLint
- TypeScript
